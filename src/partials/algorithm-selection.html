This is a Turing machine simulator. To use it:
  1. Write your Turing machine program in area. See below for syntax.
  2. Enter something in the 'Input' area - this will be written on the tape initially as input to the machine.
  3. Click on 'Run Simulation' to start the Turing machine.

{{inputList}}

Syntax:
  • Each line should contain one tuple of the form '<current state> <current symbol> <new symbol> <direction> <new state>'.
  • You can use numbers for <current state> and <new state>. Ex: 0, 1, 2 ...
  • The accept state is represented for "A" and the reject state is represented for "R"
  • You can use 0, 1, #, x or y for <current symbol> and <new symbol>. Symbols are case-sensitive.
  • <direction> should be 'L', 'R' , denoting 'move left' or 'move right', respectively.
  • The machine stop when de current state is A or R.
Exemple:

w = {01*0}

0 0 x R 0

0 1 x R 1

1 0 x R 2

1 1 1 R R

2 B x _ A

2 0 x _ R
